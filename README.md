# An AI-Based Resume Skill Gap Detection System

## 📝 Description

This project helps job seekers understand what skills they're missing compared to job requirements. It uses AI to analyze resumes and job descriptions, then shows you which skills you should learn to land your dream job. Think of it as a personal career coach that tells you exactly what to work on!

## ✨ Features

- Upload your resume and get analyzed
- Enter a job description to see what skills are needed
- Get a list of skills you're missing
- See recommendations for what to learn
- Simple and easy-to-use interface
- Powered by AI to give accurate results

## 🛠️ Technology Stack

- **V0.dev** - For building the user interface quickly
- **AI Models** - To understand resumes and job requirements
- **Embeddings** - To compare skills and find gaps
- **Basic UI** - Clean, beginner-friendly design
